# 🎬 Movie Search Application

A responsive and interactive Movie Search Application developed using **HTML5, CSS3, and JavaScript**. This project was completed as part of my **Web Developer Internship at Nexsoft Solutions**.

The application integrates with a public movie API to allow users to search for movies, view posters and details, and navigate through search results using pagination.

---

## 📌 Project Overview

The Movie Search Application enables users to explore movie information in a simple and user-friendly interface. By leveraging API integration, the application dynamically fetches and displays movie data based on user searches.

---

## ✨ Features

### 🔍 Movie Search

* Search for movies by title.
* Fetch and display results in real time.

### 🎥 Movie Details

* Display movie posters, titles, release years, and additional information.
* View detailed information through an interactive interface.

### 📄 Pagination

* Navigate through multiple pages of movie results efficiently.
* Improve usability when handling large datasets.

### ⏳ Loading Indicator

* Display a loading spinner while fetching data from the API.

### ⚠️ Error Handling

* Show friendly messages when movies are not found.
* Handle API failures and network-related issues gracefully.

### 📱 Responsive Design

* Optimized for desktop, tablet, and mobile devices.
* Provides a consistent user experience across screen sizes.

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/movie-search-app.git
```

### 2. Open the Project

Navigate to the project directory and open it in your preferred code editor.

### 3. Configure the API Key

Obtain a free API key from the movie API provider and replace the placeholder key in `script.js`.

```javascript
const API_KEY = "YOUR_API_KEY";
```

### 4. Run the Application

Open `index.html` in your browser or use the Live Server extension in Visual Studio Code.
